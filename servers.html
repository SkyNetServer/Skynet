How to join

  1. Home
  2. Server

Step 1

Open your minecraft bedrock. log-in to your Xbox live account. after log-in to your Xbox live account now its time to add the server.

Step 2

Now it's time to add the server as i told you before. click on the play button and then go to servers tap. Now we need to add the server ip and port

Step 3

Time to join! you can add a new server by clicking on the add Server button. You can put any name you like. The server IP need to be: play.skynetmc.tk and then the port is: 25668 after putting all of that. click on save. The server will show up on your servers list
